Prime Number Generator
Author: Jacob Grealy
Date 2/23/2017

Main Console Program: \PrimeNumberGeneratorEclipseProject\src\Implementation\PrimeFactorConsoleApplication.java
Requires the following:
Interface: \PrimeNumberGeneratorEclipseProject\src\Implementation\PrimeNumberGenerator.java
Concrete Class: \PrimeNumberGeneratorEclipseProject\src\Implementation\JacobsPrimeNumberGenerator.java

The PrimeFactorConsoleApplication.java takes exactly 2 parameters: integer startingValue, integer endingValue.  
It will generate and print to the console the prime numbers in the range between and including these two values. For example: 7900 and 7920 will result in 7901, 7907, 7919.
